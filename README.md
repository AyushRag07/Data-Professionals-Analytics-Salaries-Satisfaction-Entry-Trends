# 📊 Data Professionals Analytics Dashboard

## 🚀 Project Overview

An interactive **Power BI dashboard** analyzing global survey data of data professionals to uncover insights into **salaries, job satisfaction, skills, and entry barriers** in the data industry.

This project focuses on transforming raw survey data into **actionable insights** for career decision-making and industry understanding.

---

## 🎯 Business Objective

To analyze the **current state of data careers** by answering key questions:

* Which roles offer the highest salaries?
* How satisfied are professionals with their jobs?
* How difficult is it to break into data roles?
* Which skills are most in demand?

---

## 📊 Key Insights

* **Data Scientists** earn the highest average salaries among surveyed roles
* **Work-life balance satisfaction** is higher than salary satisfaction
* A significant portion of respondents find it **moderately difficult to break into data**
* **Python** is the most preferred programming language
* Majority of respondents are from **US, India, and UK**

---

## 📈 Dashboard Highlights

* Role-based salary comparison
* Job satisfaction analysis (Salary vs Work-Life Balance)
* Entry difficulty distribution
* Programming language trends
* Country-wise workforce distribution
* KPI cards for quick insights

---

## 🛠️ Tech Stack

* **Power BI** – Dashboard & data visualization
* **Power Query** – Data cleaning & transformation
* **Excel** – Data source

---

## 🧹 Data Preparation

* Cleaned missing and inconsistent values
* Standardized job titles and categories
* Converted salary data for consistency
* Structured dataset for analysis

---

## 💡 Key Takeaways

This analysis highlights that while data careers offer **high earning potential**, challenges such as **entry barriers and varying satisfaction levels** still exist. Skill selection, especially programming languages, plays a crucial role in career growth.
